資安快訊!SQL Server將來到期，建議所有用戶升級

2022-07-04 文/林妍溱

微軟上周提醒，SQL Server 2012將在今年7月12日來到支援周期終點（end of support，EoS），而Windows Server 2012/2012 R2則會在明年10月10日結束支援。兩項產品微軟提供付費延伸安全更新服務。 Windows Server 2012是在同年10月30日推出，2018年10月終止主流支援。SQL Server 2012則在同年5月20日推出，2017年7月主流支援結束。微軟警告，在公佈日期之後，這些產品將不會再接獲安全更新、非安全的功能更新、功能修補或技術支援。微軟建議用戶升級到Windows Server 2022及SQL Server 2019。 用戶若具備大量授權（Enterprise Agreement）下的軟體保證（Software Assurance），則可以購買Windows 2012、2012 R2及SQL Server 2019的延伸安全更新（ESU）。針對這些產品，微軟提供付費的延伸安全更新，最長可選擇3年服務。

不過微軟更希望用戶轉移到Azure雲端，建議用戶儘速升級應用程式和工作負載到Azure版服務，像是Azure SQL Managed Instance，或是透過負載平移（lift-and-shift）方式，將應用程式轉到Azure Virtual Machines包括Azure Dedicated Host、Azure VMware Solution和Azure Stack（Hub、HCI、 Edge）以免費獲得3年ESU。 此外，如果用戶之前買過SQL Server 2008/2008 R2和Windows Server 2008/2008 R2的ESU，微軟提醒，這些服務將分別在2022年7月12日，以及2023年1月10日終止。屆時，企業若將這些環境搬到Azure上，則可再獲得1年免費的ESU。 微軟2周前也警告Windows 8.1將在明年1月10日終止所有支援，但不提供任何付費支援，即ESU。
